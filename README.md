## File Structure

- app
    - controllers/
        - State management & controllers
    - data/
        - Data and constants
    - provider/
        - API calls
    - theme/
        - Holds theme colors and variables
    - widgets/
        - Reusable UI components
    - views/
        - Holds views and screens
        - */widgets/
            - Non-reusable UI components specific to the screen
    - models/
        - Contains app data models
